# HandiFix

## 简介

HandiFix 是一个用于批量创建 X 定时推文功能的脚本，模拟输入推文内容和时间。

守护每位书局人的双手，远离腱鞘炎。

## 使用方法

1. 安装 Tampermonkey 或类似的浏览器扩展。
2. 将 `main.js` 文件的内容复制到 Tampermonkey 中。
3. 打开 X 网站，点击 Tampermonkey 扩展中的 **守护每位书局人的双手** 脚本。
4. 按照提示输入推文内容和时间，点击确认按钮即可。
